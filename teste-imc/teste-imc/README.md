### Gerar ambiente virtual

```cmd
python -m venv venv
```

### Abrir o ambiente virtual

```cmd
venv\Scripts\activate.bat
```

### Instalar o coverage

```cmd
pip install coverage
```

## Simulação

### Executar testes

```cmd
coverage run <ARQUIVO>
```

### Ver estatísticas

```cmd
coverage report
```

### Gerar HTML

```cmd
coverage html
```
